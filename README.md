# VDesktopWheelSwitcher
### Автоматизация комбинаций управления виртуальными рабочими столами<br><br>

* `Win+Ctrl+D` - Создание
* `Win+Ctrl+(←/→)` - Переключение
* `Win+Ctrl+F4` - Удаление

Вместо этого теперь как на Ubuntu можно переключаться скроллингом по панели задач<br><br>
P.S. В Ubuntu это появилось с версии 11.04 (2011 год), я был крайне удивлён при переходе на Windows тому, что здесь этого нет даже в виде приложения
